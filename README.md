## 技术栈
前端： vue2,es6
后端： express

项目预览地址：linyouhang.cn/forum


## Project setup

为确保项目正常运行,请先运行前端再运行后端
# 1.
```
cd vue-forum-sever
```
```
npm install
```
```
node app.js
```
# 2.
```
cd vue-forum
```
```
npm install
```
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
